Chapter 1_Introduction

Date Communications

Components

Telecommunication(電話通訊) v.s. Date communication(資料通訊)

  • telecommunication:
    電話接通後，電話兩端不可再有人同時打入電話。

  • date communication:
    達到可以多方傳遞，或只同時傳遞。

Five components of date communication:
  1. Sender(寄送人):
    The sender is the device that sends the data message.

  2. Receiver(接收人):
    The receiver is the device that receives the message.

  3. Message(訊息):
    The message is the information (data) to be communicated.

  4. Transmission medium(傳輸介質):
    The transmission medium is the physical path by which a message travels from sender to receiver.

  5. Protocol(協定):
    A protocol is a set of rules that govern data communications.

Data Flow

Communication between two devices:

  • Simplex:
    The communication is unidirectional.(一個只傳送訊息，另一個只接收)

  • Half-duplex:
    Each station can both transmit and receive, but not at the same time. When one device is sending, the other can only receive.(都可以傳送和接收，但不能兩邊同時做)

  • Full-duplex(duplex):
    Both stations can transmit and receive simultaneously.(兩邊可以同時發送與接收)
